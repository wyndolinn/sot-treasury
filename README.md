# Treasury

With this app, you can calculate the total value of your treasures while taking emissary bonuses into account. It uses a remote database and only syncs treasure when you choose to do so.

### Key Features
- Calculate the total value of your treasures
- Select an emissary and its grade to increase the treasure value
