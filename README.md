# Treasury
This app is designed as a companion tool for **Sea of Thieves** and allows you can estimate the value of your loot while taking emissary bonuses into account.

The app works fully offline after the initial data load, and updates are only applied when manually triggered by the user.

### Features
- Calculate the total value of your treasures  
- Select an emissary and apply grade-based bonuses  
- Manual data updates to fetch the latest information  
- Full offline support after the initial setup
